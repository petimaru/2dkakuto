# Pixel Pro Wrestling

スマホ縦持ち向けの2Dプロレス格闘ゲームです。

## iPhoneでテストする方法

MacとiPhoneを同じWi-Fiにつないでください。

このフォルダでローカルサーバーを起動します。

```bash
python3 -m http.server 8000
```

この環境ではMacのWi-Fi IPは次の値です。

```text
192.168.0.49
```

iPhoneのSafariで次を開きます。

```text
http://192.168.0.49:8000/
```

もし開けない場合は、MacのIPが変わっている可能性があります。そのときは次のコマンドで確認します。

```bash
ifconfig en0
```

`inet` の右に出る `192.168...` の数字を使います。

## ローカル対戦サーバーを試す方法

オンライン対戦の接続テスト用に、ローカル専用の待ち合わせサーバーを起動できます。

```bash
node server.js
```

起動すると次のURLで待ち受けます。

```text
http://localhost:8787/
```

このサーバーは8桁コードの部屋と接続メッセージだけを一時的に預かります。古い部屋は約10分で自動削除されます。

## 操作

- 画面左側ドラッグ: リング内を上下左右に移動
- 画面右側タップ: パンチ
- ダッシュ中に画面右側タップ: ラリアット
- 画面右側上スワイプ: 上段蹴り
- 画面右側下スワイプ: 下段蹴り
- 相手方向スワイプ: 掴み
- 相手と逆方向スワイプ: 防御

掴み中:

- タップ: ロープに投げる
- 上スワイプ: ブレーンバスター
- 下スワイプ: 四の字固め
- 相手方向スワイプ: 頭突き
- 相手と逆方向スワイプ: バックドロップ

四の字固め中は画面右側を連打します。

## スタミナ

画面下の `ST` はスタミナです。攻撃を出すたびに減り、0になると数秒間疲れて動けなくなります。スタミナは時間とともに回復します。

## グラフィック素材ルール

キャラクター素材は全員、右向きで作ります。ゲーム内では向きに合わせて自動で左右反転します。
